PhoneGap for Windows CheatSheet

Under PhoneGap 

Install Java run time
http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

Install git
http://git-scm.com/download/win

Install apache ant
http://ant.apache.org/
http://ant.apache.org/manual/index.html

Install Android SDK
http://developer.android.com/sdk/index.html#download

Via the command line set the following variables (Modify to suit your own paths)
C:\>set ANT_HOME=C:\Ant\apache-ant-1.9.4
C:\>set JAVA_HOME=C:\Program Files\Java\jdk1.8.0_05
C:\>set PATH=%PATH%;C:\Ant\apache-ant-1.9.4\bin
C:\>set ANDROID_HOME=C:\AndroidSDK\sdk\
C:\>set PATH=%PATH%;C:\AndroidSDK\sdk\platform-tools
C:\>set PATH=%PATH%;C:\AndroidSDK\sdk\tools

Via the command line test ANT is installed correctly
C:\>ant
Buildfile: build.xml does not exist!
Build failed

Via the command line open the Android Virtual Device Manager 
C:\>android avd
Follow the steps from creating a virtual Device
Close the Window


Install node.js
http://nodejs.org/

Install PhoneGap, Create and Build and App
Via the command line complete the following steps
C:\>npm install -g phonegap
C:\>npm install -g cordova
C:\>cd D:\PhoneGap
C:\>d:
D:\PhoneGap>cordova create hello com.example.hello HelloWorld
D:\PhoneGap>cd hello
D:\PhoneGap\hello>cordova platform add android
D:\PhoneGap\hello>cordova platforms ls
D:\PhoneGap\hello>cordova build
D:\PhoneGap\hello>cordova emulate android

http://docs.phonegap.com/en/3.5.0/guide_cli_index.md.html#The%20Command-Line%20Interface
